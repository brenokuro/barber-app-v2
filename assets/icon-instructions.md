# Ícone para APK

Para criar um ícone simples, você pode:

1. **Usar ferramenta online**: 
   - Acesse https://favicon.io/favicon-generator/
   - Crie um ícone simples com "B" ou tesoura
   - Baixe os arquivos e coloque em /assets/

2. **Ícone temporário**:
   - O Expo usará ícone padrão se não encontrar
   - Build funcionará normalmente

3. **Recomendação**:
   - Tesoura + "B" para barbearia
   - Fundo dourado/preto
   - Tamanho 1024x1024px
